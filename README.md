# Product Preview Card Component

A responsive **Product Preview Card** built from the *Frontend Mentor* challenge.  
This project demonstrates building a UI card that displays a product with image, details, prices, and a call‑to‑action — using only HTML and CSS. :contentReference[oaicite:1]{index=1}

## 📌 Overview

This repository contains a product card component with:

- Product image and info
- Category, title, description
- Current and old pricing
- Add to Cart button
- Mobile‑first responsive layout

The component adjusts layout for various screen sizes and includes hover and focus states for interactive elements. :contentReference[oaicite:2]{index=2}

## 🧠 Built With

- HTML5  
- CSS3  
- Responsive design techniques
